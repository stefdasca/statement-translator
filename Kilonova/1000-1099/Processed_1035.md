
# Input data

The first line contains a natural number $n$.

# Output data

The first line will contain a natural number $s$.

# Constraints and clarifications

* $1 \leq n \leq 10^{18}$
* $n \leq s \leq 2^{60}$ 

# Example 1

`stdin`
```
500
```

`stdout`
```
512
```

## Explanation

-Give me 500 lei as a loan...
-Here is 512, to make it a round sum...
