
# Input data

The first line in `stdin` contains a natural number $n$.

# Output data

The first line in `stdout` contains a natural number $d$.

# Constraints and clarifications

* $1 \leq n \leq 366$
* $1 \leq d \leq 31$
* Tests for this problem are grouped.
